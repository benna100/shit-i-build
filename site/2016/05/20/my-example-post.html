Jekyll the static site generator

So Jekyll is this very cool tool that creates static pages using ruby. Think of it as a CMS without all of the pain (well okay there was a bit of pain, but not compared to a classic CMS). Obvioulsly its not as powerful as a fullblown CMS, but still really cool if you want to do simpler stuff. It works by using template files to create actual static pages, a little bit like Handlebars. So input is template files and posts written in a language called Markdown and out comes a generated static page.
